# DogGuessr

Браузерная игра, в которой нужно угадывать породы собак по фотографиям.

За одну игру вам показывают 10 фотографий собак. Ваша задача — найти и выбрать правильную породу на большой интерактивной карте. Чем ближе выбранная вами порода к правильному ответу, тем больше баллов вы получаете.

Поиграть можно здесь: https://dogguessr.ru/

В игре 389 различных пород. Исходные фотографии (около 37 тысяч снимков) взяты из открытого датасета: https://huggingface.co/datasets/benni-ben/dog-breeds

Карта пород сгенерирована на основе комплексной похожести собак (внешность, родство и другие характеристики). Алгоритм укладывает эти данные в прямоугольную сетку, из-за чего родственные породы естественным образом образуют на карте своеобразные материки.

Очки за каждый раунд зависят от расстояния на карте между вашей догадкой и правильным ответом. Чтобы оценка оставалась точной, формула начисления баллов дополнительно компенсирует искажения, которые неизбежно возникают при переносе многомерных связей между породами на плоскую 2D-карту.
